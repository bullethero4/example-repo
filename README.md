# Album Management

A Python program that demonstrates OOP and list manipulation with an `Album` class.

What it does:
- Creates an `Album` class with name, artist, and number of songs
- Displays a list of albums
- Sorts albums by number of songs in ascending order
- Swaps album positions in a list
- Finds the index of a specific album by name
- Copies, extends, and sorts a second list of albums alphabetically
